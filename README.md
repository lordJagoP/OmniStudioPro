## 🔊 OmniStudio — Browser-Based Audio Mastering Suite

**OmniStudio** is a professional-grade, browser-based audio mastering engine built entirely with the **Web Audio API** — zero plugins, zero installs.  
It delivers real-time EQ shaping, loudness processing, spectrum feedback, and profile-driven automation directly inside a modern web workflow.

> **Core Features**
> - 🎚️ Multi-band parametric EQ with live frequency response curve
> - 📊 Real-time spectrum analyzer + LUFS/RMS loudness meter
> - 🎛️ Preset profiles *(Podcast · Vinyl · Club · Broadcast · Film · Custom)* — **EQ auto-simulates on profile change**
> - 🔁 A/B instant preview — dry vs. processed, with smooth 50–200 ms parameter ramping (no zipper noise)
> - 🗜️ Multiband compressor with sidechain support
> - 📁 Client-side WAV / MP3 export via Web Workers

**Stack:** `Web Audio API` · `AudioWorklet` · `Canvas API` · `Web Workers` · `JavaScript`
